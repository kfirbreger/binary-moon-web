# Web assets

Files referenced from `web/index.html`. Drop your real assets here using
these filenames and the page picks them up.

| Filename | What | Notes |
|---|---|---|
| `screen-patch.png` | Patch view screenshot | ~600×1300 portrait, PNG. Trimmed to phone frame or bare. |
| `screen-phases.png` | Phases view screenshot | Same dimensions. |
| `screen-pad.png` | Performance pad screenshot | Same dimensions. |
| `demo.mp3` | ~30s audio loop | MP3 at 128-192 kbps stereo. Mono is also fine for an ambient drone. |
| `app-store-badge.svg` | "Download on the App Store" badge | Get the official asset from Apple's marketing resources page. Keep the SVG so it scales. |
| `favicon.png` | Browser tab icon | 32×32 or 64×64 PNG. |
| `og-image.png` | OpenGraph share image | 1200×630 PNG. Used when the link is shared on Mastodon, Slack, iMessage, etc. |

The page works with missing assets — broken `<img>` tags will just show
alt text. Drop them in as you have them.
